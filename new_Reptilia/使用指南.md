# Ask4Porn 爬虫程序使用指南

## 🎯 项目概述

本项目是针对 https://ask4porn.cc/ 网站的自动化爬虫程序，具备以下特性：

- ✅ **Cookie自动获取** - 基于Playwright的自动化Cookie提取
- ✅ **Cloudflare防护绕过** - 解决"Just a moment..."验证页面
- ✅ **高质量视频选择** - 自动选择分辨率最高的视频
- ✅ **广告视频过滤** - 自动过滤小于10MB的视频（可能为广告）
- ✅ **单一/批量下载** - 支持单个视频下载和批量下载
- ✅ **异步下载** - 高效的异步下载机制

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
playwright install chromium
```

### 2. 获取Cookie（首次运行）

```bash
python get_cookies.py
```

这个脚本会：
- 自动安装Playwright浏览器
- 访问目标网站并处理Cloudflare验证
- 提取Cookie并保存到cURL.txt文件

### 3. 运行主程序

```bash
python main.py
```

## 💡 使用说明


#### 2. 批量下载所有媒体

#### 3. 测试连接

#### 4. 配置设置
- 查看当前配置参数

#### 5. 查看帮助
- 显示详细使用说明
- 包含故障排除指南

### Cookie管理

#### 自动获取（推荐）
```bash
python get_cookies.py
```

#### 手动配置
1. 在浏览器中访问目标网站
2. 按F12打开开发者工具
3. 复制cURL命令到cURL.txt文件
4. 确保config.py中的curl_file_path指向正确文件

### 配置参数

编辑 `config.py` 可以调整以下参数：

```python
# Cookie配置
self.curl_file_path = "cURL.txt"  # Cookie文件路径

# 下载配置
self.download_dir = "downloads"   # 下载目录
self.max_concurrent = 1          # 最大并发数
self.max_retries = 3             # 最大重试次数
self.timeout = 30                # 超时时间（秒）

# 视频质量配置
self.video_quality_config = {
    'ad_filter_enabled': True,                    # 启用广告过滤
    'ad_file_size_threshold': 10 * 1024 * 1024,  # 10MB阈值
    'prefer_formats': ['mp4', 'webm'],            # 优先格式
}
```

## 🔧 故障排除

### 1. Cookie问题

**症状**: 访问网站返回403或"Just a moment..."

**解决方案**:
```bash
# 重新获取Cookie
python get_cookies.py

# 或者手动更新cURL.txt文件
# 访问目标网站，复制新的cURL命令
```

### 2. Cloudflare验证问题

**症状**: 持续看到Cloudflare验证页面

**解决方案**:
- 使用 `get_cookies.py` 重新获取Cookie
- 确保浏览器成功通过验证
- 检查网络连接稳定性

### 3. 下载问题

**症状**: 下载失败或中断

**解决方案**:
- 检查网络连接
- 增加timeout值
- 减少max_concurrent值
- 启用代理（如果需要）

### 4. 编码问题

**症状**: Brotli解码错误

**解决方案**:
```bash
pip install brotli
```

### 5. 依赖问题

**症状**: 模块导入错误

**解决方案**:
```bash
# 重新安装依赖
pip install -r requirements.txt
playwright install chromium
```
